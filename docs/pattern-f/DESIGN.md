# DESIGN.md — Pattern F「ストーリーテリング」

## Design Concept
charity: water やエディトリアルサイトのように、写真・言葉・余韻で
活動の意味を物語として伝えるデザイン。OIKOSの価値を説明するのではなく、
子どもと大学生が出会う瞬間、場が生まれる変化、その先の未来を章立てで感じさせる。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #8C4A32 | 焦げたシナモン（物語の熱） |
| --secondary | #355C5A | 深い青緑（静かな希望） |
| --accent | #D9A25F | 光のようなゴールドアクセント |
| --bg | #F7F1E8 | 物語を包む温かい背景 |
| --bg-deep | #2A2623 | 夜のようなダーク背景 |

| --line | rgba(46, 41, 37, 0.14) | 区切り線 |

## Typography
- **見出し**: Shippori Mincho（情緒と余韻）
- **本文**: Noto Sans JP（可読性重視）
- **英語見出し**: EB Garamond（エディトリアル感）
- **サイズスケール**: 14px / 16px / 18px / 26px / 38px / 56px / 76px

## Layout Principles
- 最大幅1160px、章ごとに情緒のある流れをつくる
- 写真を強く使い、引用・数字・短い文章で物語をつなぐ
- スクロールによって「課題→出会い→変化→参加」を体験させる
- 明暗の切り替えで感情の波を生む

## Component Style
- ボタン: 少し厚みのある角丸、物語の終端で行動を促す
- カード: 写真や引用を主役にした誌面風レイアウト
- セクション見出し: 章番号、英語の前振り、日本語大見出し
- 数値実績: 単なる実績でなく、変化の文脈付きで提示する

## Mood
夕方の帰り道 / ノートの余白に残る言葉 / 誰かの記憶に残る声
