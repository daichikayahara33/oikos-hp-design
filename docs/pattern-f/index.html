Story begins here

帰る場所が増えると、
明日を待つ気持ちも増えていく。

OIKOSがつくるのは、子どもたちと大学生が出会い、同じ時間を重ねる場所。この案では、活動を説明するのではなく、そこに生まれる変化を物語として追体験できるトップページを目指します。

物語を読む 次の章へ参加する
課題 → 出会い → 変化 → 参加 の流れで、OIKOSの価値を体験として見せる。

「また来るね」と言って帰っていく子の背中を見送るたびに、この場所には意味があると思う。

University student / youth spot
Why we start

学校や家庭とは別に、安心していていい場所が必要だった。

子どもが何かを頑張るための場所ではなく、ただそのまま来ていい場所。OIKOSの活動は、そうした空白に小さな灯りを置くところから始まります。

「何も話さなくても、ここに座っていていいと思えた日があった。」

この案では、理念や事業説明をひと息で読ませるのではなく、写真と短い言葉をつなぎながら、なぜこの活動が必要なのかを静かに伝えていきます。

What happens here

出会って、話して、少しずつ安心が積み重なっていく。

居場所の時間は、大きな出来事ではなく、小さなやり取りの連続でできています。大学生が近い存在としてそこにいることが、子どもたちにとっての安心につながります。

まずは来てみる

勉強しても、遊んでも、話さなくてもいい。自分のペースでいられる入口があります。

顔なじみが増える

何度か通ううちに、学生スタッフや他の子どもたちとの関係が少しずつ育っていきます。

明日を考えられるようになる

安心できる場所があることで、学校や日常のことを少し前向きに見られるようになることがあります。

Support

この物語の続きは、支えてくれる人と一緒に書いていく。

寄付、連携、ボランティア参加。OIKOSの居場所づくりは、誰かひとりの力ではなく、関わる人の想いが重なって続いていきます。最後に行動を促すセクションも、強い販促ではなく、余韻のある呼びかけとして設計します。
