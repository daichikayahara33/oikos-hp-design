# DESIGN.md — Pattern D「静かな洗練」

## Design Concept
無印良品やNotionに通じる、静けさと上質さを重視したデザイン。
やさしさを残しながらも装飾を抑え、余白・紙のような質感・整ったタイポグラフィで
OIKOSの誠実さと持続的な活動の深みを表現する。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #2F4F4A | 深いセージグリーン（軸色） |
| --secondary | #7B8C87 | 青みのあるグレージュ（補助色） |
| --accent | #C98E5B | 静かなアンバー（CTA、見出しアクセント） |
| --bg | #F8F6F1 | 紙のようなオフホワイト背景 |
| --bg-soft | #EFEBE3 | 淡いセクション背景 |
| --text | #2B2A28 | 墨に近い本文色 |
| --text-sub | #736E68 | 落ち着いたサブテキスト |
| --line | rgba(43, 42, 40, 0.12) | 細い区切り線 |

## Typography
- **見出し**: Noto Serif JP（静かな格調）
- **本文**: Noto Sans JP（素直で読みやすい）
- **英語見出し**: Cormorant Garamond（繊細で余韻のある表情）
- **サイズスケール**: 14px / 16px / 18px / 24px / 36px / 52px / 68px

## Layout Principles
- 最大幅1180px、ゆったりした余白で呼吸感をつくる
- セクションごとに紙面のような静かなリズムを持たせる
- 写真は大きく使うが、角を取りすぎず端正に見せる
- 情報量を抑え、見出し・写真・短い文章で余白を活かす

## Component Style
- ボタン: 横長、角丸控えめ、細いボーダーと静かなホバー
- カード: 罫線ベース、シャドウ最小、紙片のような印象
- セクション見出し: 英語小見出し + 日本語大見出し、余白を大きく取る
- 数値実績: 大きな数字よりも静かな整列感で見せる

## Mood
朝の図書館 / 生成りの紙 / 静かに差し込む自然光
