# DESIGN.md — Pattern B「凛」

## Design Concept
モダンで洗練されたミニマルデザイン。余白と書体の力で信頼感を表現。
カタリバやフローレンスのような「社会を変える」強いメッセージ性を持ちつつ、
ノイズを削ぎ落とした知性的なデザイン。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #1A1A2E | ディープネイビー（メイン） |
| --primary-accent | #E07B4C | テラコッタオレンジ（CTA） |
| --secondary | #4A7C6F | ティールグリーン（サブアクセント） |
| --bg | #FFFFFF | クリーンホワイト |
| --bg-dark | #1A1A2E | ダークセクション背景 |
| --text | #1A1A2E | ネイビーテキスト |
| --text-sub | #6B7280 | グレーサブテキスト |
| --border | #E5E7EB | 薄いボーダー |

## Typography
- **見出し**: Noto Sans JP Bold（力強く現代的）
- **本文**: Noto Sans JP Regular
- **英語見出し**: Inter（クリーンで知的）
- **サイズスケール**: 14px / 16px / 18px / 24px / 36px / 56px / 72px

## Layout Principles
- 最大幅1280px、ダイナミックな余白（セクション間100px〜160px）
- ヒーローはフルスクリーン、大胆なタイポグラフィ
- ダーク/ライトセクションの交互切り替えでリズム
- グリッドを効かせた整然とした情報配置

## Component Style
- ボタン: 角丸8px、シャープ、ホバーで背景色反転
- カード: 角丸8px、ボーダーライン、ミニマルシャドウ
- セクション見出し: 大きな英語 + 日本語を重ねるレイヤー表現
- 数値実績: 超大型フォント + 横並び

## Mood
夜明け前の静けさ / コンクリートと植物 / 凛とした決意
