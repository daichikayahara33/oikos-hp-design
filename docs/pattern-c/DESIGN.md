# DESIGN.md — Pattern C「つながり」

## Design Concept
カラフルでエネルギッシュ、でも品のある楽しさ。むすびえの温かさと
charity: waterの洗練を掛け合わせた、行動を促すデザイン。
「大学生×子ども」の若いエネルギーを視覚的に表現する。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #FF6B35 | ビビッドオレンジ（エネルギー） |
| --secondary | #2EC4B6 | ターコイズ（爽やかさ・つながり） |
| --tertiary | #FFD166 | サンフラワーイエロー（子どもの明るさ） |
| --bg | #FEFEFE | ほぼ白 |
| --bg-warm | #FFF8F0 | 温かい白 |
| --bg-cool | #F0FAFA | 涼しい白 |
| --text | #1B2432 | ダークネイビー |
| --text-sub | #64748B | スレートグレー |

## Typography
- **見出し**: Zen Maru Gothic（丸ゴシックで親しみ）
- **本文**: Noto Sans JP（可読性）
- **英語見出し**: Quicksand（丸みと現代感の両立）
- **サイズスケール**: 14px / 16px / 20px / 28px / 40px / 56px

## Layout Principles
- 最大幅1200px、リズミカルな余白
- 背景色の切り替え（warm / cool / white）でセクション区別
- イラスト的な装飾要素（丸、波線）で遊び心
- カードのグリッド配置で情報をスッキリ見せる

## Component Style
- ボタン: 角丸全丸（pill型）、ホバーでスケールアップ
- カード: 角丸20px、カラフルなトップボーダー、ホバーで浮く
- セクション見出し: カラーアクセント付き + アイコン
- 数値実績: カラフルな背景カード + アニメーション

## Mood
放課後の校庭 / カラフルなクレヨン / みんなの笑い声
