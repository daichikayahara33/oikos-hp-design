Vision

すべての子どもたちが
明日を楽しみにできる
社会をつくる

学校と家庭の間に、安心して通い続けられる居場所が不足しています。 気軽に話せる年上の存在がいないまま、困りごとが見えにくく進んでしまうこともあります。 OIKOSは、制度の手前にいる子どもたちに向き合い、関係が続く居場所を育てます。

01

「またね」と言えるオフラインの居場所をふやす

継続して通える場をつくり、子どもたちの日常に安心の拠点を増やします。

02

ナナメ上の学生とのつながりを増やす

年齢の近い大学生だからこそ生まれる、自然で話しやすい関係を大切にします。

03

ありのままでいいと思える子どもを増やす

自分のままで過ごせる時間と関係を通じて、自己肯定感を育みます。

Key Concept

「ゆーすぽっと」とは

大学生など若い担い手が、子どもたちとの距離の近さを活かしてつくる居場所です。 食事・遊び・学び・おしゃべりを通して、子どもたちがほっとできる時間を届けます。

Activities

3つの活動

居場所の実践から、他拠点への伴走、ネットワーク形成まで。OIKOSは子どもたちを支える仕組みを広げています。

ACTIVITY 01

ゆーすぽっとの
設計と運営

大学キャンパスや地域の既存資源を活用した、学生主体の子どもの居場所。主に10代までの子ども・若者を対象に、食事・遊び・学び・おしゃべりの場をつくります。

東京大学 東京科学大 食事・遊び
ACTIVITY 02

他拠点の立ち上げ・
伴走支援

立ち上げ設計、運営マニュアル提供、研修、安全基準の策定、引き継ぎ体制の構築まで伴走。OIKOSなしでも自走できる状態を目指します。

立ち上げ支援 研修 伴走
ACTIVITY 03

ネットワーク形成・
調査・発信

地域・行政・大学・NPO・企業との連携を広げ、勉強会や会議を開催。知見を集め、届け、次の居場所づくりにつなげます。

連携 調査 発信

Message

古川 陽登

代表理事 / 大学3年生

原点は、2年前に立ち上げた荒川区の子ども食堂でした。そこで気づいたのは、食事以上に、安心して過ごせる時間と関係が子どもの心を支えているということです。

学生である今だからこそ、子どもたちとの距離が近い。その強みを活かして、持続可能な「居場所」の仕組みを創りたいと考えています。

すべての子どもたちが「明日が楽しみだ」と思える社会を、一緒につくっていけたらうれしいです。
