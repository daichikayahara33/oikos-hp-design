# DESIGN.md — Pattern A「ぬくもり」

## Design Concept
温かみと信頼感を重視。Airbnb + Notionのエッセンスを取り入れた、
やわらかく安心感のあるデザイン。子どもの居場所をつくるNPOとして、
サイト自体が「居心地の良い場所」であることを体現する。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #E8713A | メインオレンジ（CTA、アクセント） |
| --primary-light | #FFF4ED | オレンジの淡い背景 |
| --secondary | #5B8C6B | やわらかグリーン（信頼・成長） |
| --bg | #FFFBF7 | クリーム系オフホワイト背景 |
| --bg-section | #FDF6EE | セクション交互背景 |
| --text | #2D2A26 | ダークブラウン（温かみのある黒） |
| --text-sub | #7A7570 | サブテキスト |
| --accent | #D4A574 | ゴールドベージュ |

## Typography
- **見出し**: Noto Serif JP（明朝体で上品さ・信頼感）
- **本文**: Noto Sans JP（可読性重視）
- **英語見出し**: Playfair Display（エレガント）
- **サイズスケール**: 14px / 16px / 20px / 28px / 40px / 56px

## Layout Principles
- 最大幅1200px、たっぷりの余白（セクション間80px〜120px）
- 写真を大きく使い、角丸16pxでやわらかさを演出
- カード型レイアウトで情報を整理
- スクロールで「課題→解決→参加」のストーリーを語る

## Component Style
- ボタン: 角丸24px、ホバーでやわらかく浮き上がる
- カード: 角丸16px、薄いシャドウ、ホバーで微かに浮く
- セクション見出し: 英語小見出し + 日本語大見出し
- 数値実績: 大きな数字 + 小さな説明文

## Mood
温かい朝の光 / 木のぬくもり / 手触りのある紙
