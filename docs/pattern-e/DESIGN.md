# DESIGN.md — Pattern E「信頼コーポレート」

## Design Concept
StripeやSonyのような整理感と説得力を持つ、信頼重視のコーポレートデザイン。
感情よりもまず安心感と明快さを届け、企業・大学・寄付者が
「連携先として信頼できる団体だ」と直感できる構造を目指す。

## Color Palette
| Token | Hex | Usage |
|-------|-----|-------|
| --primary | #10233F | コーポレートネイビー（基調） |
| --secondary | #0F766E | ティール（信頼・持続性） |
| --accent | #F08A4B | 明快なアクションオレンジ |
| --bg | #FFFFFF | ホワイト背景 |
| --bg-soft | #F4F7FA | 補助背景 |
| --bg-dark | #0D1B2A | ダークセクション背景 |
| --text | #132238 | 本文ネイビー |
| --text-sub | #61758A | サブテキスト |
| --line | #D8E1EA | 区切り線 |

## Typography
- **見出し**: Noto Sans JP Bold（強く明快）
- **本文**: Noto Sans JP Regular
- **英語見出し**: Manrope（現代的で精密）
- **サイズスケール**: 14px / 16px / 18px / 24px / 32px / 48px / 72px

## Layout Principles
- 最大幅1280px、整ったグリッドで秩序を出す
- ヒーローで価値提案と実績を同時に見せる
- 数値、活動領域、支援方法を明快に分節化する
- ダークセクションを要所で使い、重心と緊張感をつくる

## Component Style
- ボタン: 角丸10px、強いコントラスト、即時性のあるCTA
- カード: 直線的、薄いボーダー、制度設計のような整然さ
- セクション見出し: 英語ラベル + 大きな日本語見出し
- 数値実績: 大型タイポ、罫線、グリッド整列で説得力を出す

## Mood
役員会資料の明快さ / 深いネイビーの会議室 / 整理された信頼
